class ships.Ships(no_ships=False)¶

Class responsible for ships generation and storing

add_block_to_ship(pos, direction, is_vetrical, coords)¶

Add one block to new ship according to direction and orientation. If there if field edge just build block in opposite direction.

Parameters

  • pos – int, current x or y coordinate

  • direction – 1 (forward) or -1 (reverse)

  • is_vertical – 1 if the ship is vertical

  • coords – set of pairs (x, y) - all current blocks in ship

Returns

coordinates of chosen cell and new direction in case it was changed

choose_start(available_blocks)¶

Choose start cell for new ship.

Parameters

available_blocks – set of pairs (x, y) of all free cells in the field

Returns

coordinates of chosen cell, if the ship is vertical, if the direction is reverse
